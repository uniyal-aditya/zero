# RAFTXR — Discord Music Bot (Pure Python)

## Features
- Slash commands (play, skip, pause, resume, stop, nowplaying, queue, volume)
- Play YouTube links and search by name
- Basic Spotify link resolution (requires Spotify API keys for best results)
- Per-guild queue and player
- Modular code easy to extend
- Lavalink (wavelink) backend for stable audio playback

## Setup

1. Install Python 3.10+ (recommended)
2. Install requirements:

   pip install -r requirements.txt
